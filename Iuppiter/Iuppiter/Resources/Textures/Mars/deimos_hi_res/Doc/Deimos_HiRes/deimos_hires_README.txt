
DEIMOS HI-RES

INSTALLATION
Unzip into your Orbiter directory, make sure you preserve the directory structure.
(Warning! This will over-write the default deimos.msh, so you may wish to back it up)

INFO
Mesh = 44336 poly
Texture = 2048 x 4096 .dds


FILES INCLUDED

Textures/deimos_hires.dds
Doc/Deimos_HiRes/deimos_hires_README.txt
